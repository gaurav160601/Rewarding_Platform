# Loyalty Commerce Platform

## Overview

A scalable loyalty commerce platform where users can purchase products, earn loyalty points, redeem rewards, and manage their orders.

## User Features

### Authentication

* Register
* Login
* Logout
* Refresh Token
* Forgot Password
* Reset Password

### Product Management

* View Products
* Search Products
* Filter Products

### Cart Management

* Add To Cart
* Remove From Cart
* Update Quantity

### Order Management

* Create Order
* View Order History
* View Order Details
* Cancel Order

### Payments

* Checkout
* Stripe Payment
* Payment History

### Loyalty Program

* Earn Points
* Redeem Points
* View Reward History
* View Loyalty Tier

## Admin Features

### Product Administration

* Create Product
* Update Product
* Delete Product

### User Administration

* View Users
* View Reward Balances

### Order Administration

* View Orders
* Update Order Status

## Non Functional Requirements

* High Availability
* Horizontal Scalability
* Secure Authentication
* Event Driven Communication
* Audit Logging
* Fault Tolerance
* Distributed Architecture
